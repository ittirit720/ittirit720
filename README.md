### 🧱 The Architect Who Couldn't Code

I live in an 8-story flat in Khlong Toei, Bangkok. My background is in **Architecture**, not Computer Science.

Originally, I just wanted to write a book about an analytical framework I designed — **"RCT 7 Thinking"** (Reverse Component Thinking).

But when I discovered AI IDEs, my entire trajectory shifted. I realized I didn't need to just write a book about the framework — **I could build the AI Operating System that runs on it.**

Then on **August 11, 2025**, facing profound personal loss, I wrote the first version of the FDIA equation on paper:

```
F = D^I × A
```

> *Future = Data raised to the power of Intent, multiplied by the Architect.*  
> *When A = 0, the output is always zero — no AI can act without human authorization.*

This is not philosophy. It is a **mathematical constitutional guarantee**.
